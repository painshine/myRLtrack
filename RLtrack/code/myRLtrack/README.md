### Installation  
- This code is tested on Linux os (ubuntu 14.04) 64bit, MATLAB 2017a, and Cuda-8.0 with NVIDIA GTX 1080 TI.
- Run `adnet_compile.m` to compile MatConvNet.

### Run tracking
- Run `adnet_demo.m`.
